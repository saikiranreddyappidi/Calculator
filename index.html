App Description and User Manual

Overview

This calculator app is designed to perform both basic arithmetic and advanced logical operations, making it a versatile tool for everyday use and specialized tasks. The intuitive interface ensures that users can quickly perform calculations and manage their history with ease.

Main Features

  • Basic Arithmetic Operations: Supports addition (+), subtraction (-), multiplication (*), and division (/).
  • Immediate Results: As users type numbers and operators, the app dynamically displays the results based on the current input.
  • History Management: All completed equations are saved to the history, accessible by pressing the "HISTORY" button.
  • Logical Operations: Toggle between arithmetic and logical operations using the 'B' button. Logical operators include OR (|), AND (&), XOR (^), and NOT (~). Note that logical operations ignore decimal parts.
  • Percentage Calculation: A dedicated % button allows for easy percentage calculations.
  • History Interaction: Users can copy, share, delete individual entries, or select all history items for batch actions.
  • Privacy: Users can clear their entire calculation history using the delete button at the bottom right corner.

Detailed Description

When you open the calculator app, you will see a numpad with basic operators (+, -, *, /) and the equals sign (=). As you start typing numbers, the app immediately shows the results of the arithmetic operations based on your input. Pressing the equals sign (=) will save the current equation to the history.

At the bottom left corner, there's a button with the symbol 'B'. Pressing this button toggles the operators to logical operators: OR (|), AND (&), XOR (^), and NOT (~). These operators perform logical operations on the given input values, ignoring any decimal parts. Arithmetic operations, however, will consider floating numbers.

At the top of the screen, there are options like "HISTORY," "MORE," and "FINANCE." The "HISTORY" option lets you view all your previous calculations. The "MORE" and "FINANCE" options are still in beta versions.

In the history view, you can long-press any expression to see options like copy, share, delete, and select all. You can copy an expression, share it through apps like WhatsApp, Facebook, or Messenger, delete a specific expression, or select all expressions for batch actions. There's also a delete button at the bottom right corner to clear all history entries.

Usage Instructions

  • Performing Calculations:
    • Open the app to see the numpad and basic operators.
    • Start typing numbers and operators to see real-time results.
    • Press the equals sign (=) to save the equation to the history.
  • Switching to Logical Operations:
    • Press the 'B' button to switch to logical operators.
    • Use | for OR, & for AND, ^ for XOR, and ~ for NOT.
    • Note: Logical operations ignore decimal values.
  • Viewing and Managing History:
    • Press the "HISTORY" button to view past calculations.
    • Long-press an expression to copy, share, delete, or select all.
    • Use the delete button at the bottom right to clear all history.
  • Using Percentage Calculations:
    • Press the % button to perform percentage calculations.
